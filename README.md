Reseller Sales Performance Dashboard
تحليل شامل لأداء مبيعات الموزعين باستخدام Excel وPower BI.
نظرة عامة على المشروع
قمت بتحليل بيانات مبيعات الموزعين والتي تحتوي على أكثر من ,134 مليون مبيعات 
أرباح بإجمالي 50 مليون دولار خلال الفتره 2023 إلي 2025.
هدف المشروع كان فهم أداء المبيعات، قياس الربحية، وتحديد أوجه القوة والضعف في الأقاليم والموزعين.
التقنيات المستخدمة

Power Query لتنظيف وتحويل البيانات
Pivot Tables و Charts لبناء التحليلات
Slicers لعمل داشبورد تفاعلي
DAX لحسابات الربح والهوامش
