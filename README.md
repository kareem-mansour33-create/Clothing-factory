# Reseller Sales Performance Dashboard

تحليل شامل لأداء مبيعات الموزعين (Resellers) باستخدام Excel وPower BI.

![Dashboard Preview](dashboard-screenshot.png)

## 📋 نظرة عامة على المشروع

تحليل بيانات مبيعات أكثر من **57,800** عملية بيع بإجمالي مبيعات **77.55 مليون دولار** خلال الفترة من 2017 إلى 2018.

### 🎯 أهداف المشروع
- قياس أداء المبيعات والربحية
- تحديد أفضل الأقاليم والموزعين
- اكتشاف فرص تحسين الربحية
- بناء داشبورد تفاعلي لدعم اتخاذ القرار

## 🛠️ التقنيات المستخدمة
- **Excel** (Power Query + Pivot Tables + Charts)
- **DAX** (للحسابات المعقدة)
- **Power BI** (للنسخة التفاعلية)
- Data Cleaning & Transformation

## 📊 أهم النتائج

- **إجمالي المبيعات**: 77.55 مليون دولار
- **إجمالي الربح**: 999,495 دولار
- **هامش الربح المتوسط**: 1.29%
- وجود تفاوت كبير في أداء الأقاليم
- تركيز المبيعات على عدد محدود من الموزعين

## ⚡ أهم المشاكل والحلول

| المشكلة                          | الحل المقترح                              |
|----------------------------------|------------------------------------------|
| هامش ربح منخفض (1.29%)           | تقليل الخصومات + خفض التكاليف           |
| تفاوت أداء الأقاليم              | إعادة توزيع الموارد والدعم              |
| الاعتماد على موزعين قليلين       | تنويع قاعدة الموزعين + برامج تحفيز     |
| منتجات منخفضة الربحية            | إعادة تسعير أو إيقاف بعض المنتجات       |

## 🚀 مميزات الداشبورد

- أفضل الموزعين والمنتجات
